第三课：频域观察的窗口——离散时间傅里叶变换（DTFT）

在时域中，我们看到的是信号幅度的起伏，但难以直观看出其中包含哪些频率成分。频域分析就像一副“频谱眼镜”，戴上它，你就能看清信号的频率构成。DTFT就是为我们提供这样一副眼镜的强大工具。

1. 为什么需要频域分析？

•   时域的局限：在时域中，如果一个信号由多个频率组合而成，我们看到的只是它们叠加后的复杂波形，难以分辨具体成分。

•   频域的洞见：在频域中，信号被表示为不同频率分量的“强度图”，能清晰揭示其频谱结构。这对于滤波、压缩、识别等任务至关重要。

2. DTFT的定义：从时域到频域的桥梁

对于一个绝对可和的离散时间序列  x[n] （即满足  \sum_{n=-\infty}^{\infty} x[n]
 < \infty ），其离散时间傅里叶变换定义为：
\[
X(e^{j\omega}) = \sum_{n=-\infty}^{\infty} x[n] e^{-j\omega n}
\]
其中：
•    \omega  是数字频率（单位：弧度），它是连续变量，取值范围通常为  [-\pi, \pi]  或  [0, 2\pi] 。

•   \( e^{-j\omega n} = \cos(\omega n) - j\sin(\omega n) \) 是复正弦序列，代表一个频率为  \omega  的“基本成分”。

•   结果 \( X(e^{j\omega}) \) 是一个复数函数，包含了每个频率分量的幅度和相位信息。

直观理解：DTFT可以看作是将信号  x[n]  与无数个不同频率  \omega  的复正弦序列进行“比对”或“相关”。\( X(e^{j\omega}) \) 的值越大，说明信号中包含该频率  \omega  的成分越多。

3. 关键概念：数字频率  \omega  的含义

•    \omega  是模拟频率  \Omega （单位：弧度/秒）与采样频率  f_s  归一化的结果：

     \omega = \Omega T = 2\pi f / f_s 。
•    \omega = \pi  对应的是模拟频率  f = f_s/2 ，即奈奎斯特频率（最高可分析频率）。

•    \omega  从  -\pi  到  \pi  变化一个周期，对应模拟频率从  -f_s/2  到  f_s/2 。

4. 逆DTFT：从频域回到时域

逆变换公式为：
\[
x[n] = \frac{1}{2\pi} \int_{-\pi}^{\pi} X(e^{j\omega}) e^{j\omega n} d\omega
\]
这可以理解为：将信号  x[n]  分解为无数个幅度和相位由 \( X(e^{j\omega}) \) 决定的复正弦分量  e^{j\omega n} ，然后将所有这些分量“合成”起来，就恢复了原信号。这是一个连续的积分，对应频率的连续统。

5. 一个计算示例

考虑一个简单的矩形脉冲序列：
\[
x[n] = 
\begin{cases} 
1, & n
 \leq M \\
0, & n
 > M 
\end{cases}
\]
计算其DTFT：
\[
X(e^{j\omega}) = \sum_{n=-M}^{M} e^{-j\omega n} = \frac{\sin(\omega (M+1/2))}{\sin(\omega/2)}
\]
这个结果是一个周期性的 Dirichlet函数（离散时间的sinc函数）。其幅度谱在  \omega = 0  处有主瓣，两侧有旁瓣。随着  M  增大，主瓣变窄，旁瓣增多但幅度减小。这直观展示了时域有限脉冲的频谱是无限的，且时域越宽，频域越集中。

6. DTFT的重要性质

•   周期性：\( X(e^{j\omega}) \) 是周期为  2\pi  的周期函数。这是因为离散时间复正弦序列  e^{j\omega n}  在频率  \omega  和  \omega + 2\pi  时是完全相同的。因此，我们通常只关注一个周期，如  [-\pi, \pi] 。

•   卷积定理：时域卷积对应频域相乘，时域相乘对应频域卷积（并乘以 \( 1/(2\pi) \)）。

  \[
  x[n] * h[n] \longleftrightarrow X(e^{j\omega}) H(e^{j\omega})
  \]
  \[
  x[n] w[n] \longleftrightarrow \frac{1}{2\pi} (X(e^{j\omega}) * W(e^{j\omega}))
  \]
  这是极其重要的性质！它意味着：LTI系统在时域的卷积运算，在频域简化为乘法运算。这为滤波器设计和分析提供了极大的便利。
•   移位性质：

  •   时移：\( x[n-n_0] \longleftrightarrow e^{-j\omega n_0} X(e^{j\omega}) \)（时移对应频域的线性相移）

  •   频移：\( e^{j\omega_0 n} x[n] \longleftrightarrow X(e^{j(\omega - \omega_0)}) \)（时域调制对应频谱搬移）

•   对称性：如果  x[n]  是实序列，则其DTFT满足：

  •   幅度谱是偶函数：\( X(e^{j\omega}) = X(e^{-j\omega})
 \)

  •   相位谱是奇函数：\( \angle X(e^{j\omega}) = -\angle X(e^{-j\omega}) \)

  •   实部偶函数，虚部奇函数。

7. DTFT的局限与启示

•   局限：DTFT要求序列是无限长或绝对可和的，且变换结果是频率的连续函数，计算机无法直接存储和计算。

•   启示：这引出了我们需要一种离散的频域表示，既能用有限个数字表示，又能保持核心信息。这就是我们下一课要学的离散傅里叶变换。

本课核心：
DTFT是理解信号频域特性的理论基础。它将信号从时域映射到连续的频域，揭示了信号的频率构成，并通过卷积定理建立了LTI系统在频域的分析方法。理解DTFT的周期性、对称性以及它与模拟频率的关系至关重要。
